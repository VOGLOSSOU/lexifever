Comment utiliser ces composants

Pour utiliser ces composants réutilisables sur toutes vos pages :

Option 1 : Inclusion directe (pour les sites statiques)

  1. Copiez le code HTML du header et du footer dans des fichiers séparés (header.html et footer.html)
  2. Utilisez des techniques d'inclusion côté serveur comme PHP include, SSI, ou un système de templates
  3. Exemple avec PHP : <?php include 'header.html'; ?>

Option 2 : Composants JavaScript

  1. Créez des fichiers JavaScript qui injectent le header et le footer dans des éléments spécifiques
  2. Incluez ces scripts sur toutes vos pages
  3. Exemple : <div id="header"></div> puis utilisez JavaScript pour injecter le contenu

Option 3 : Framework frontend

  1. Si vous utilisez un framework comme React, Vue ou Angular, créez des composants Header et Footer
  2. Importez et utilisez ces composants dans toutes vos pages/vues

N'oubliez pas d'adapter les liens actifs dans la navigation en fonction de la page courante !
